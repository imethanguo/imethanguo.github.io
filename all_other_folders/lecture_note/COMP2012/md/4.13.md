## Lambda Function and Function Pointer

### Lambda Function

Lambda function is a local anonymous function (a function without a name).

```cpp
[capture-list](parameter-list) mutable -> return-type {
  // body
}
```

Example relationship with a normal function:

```cpp
void f() {}
// Similar lambda form:
[]() {}
```

A lambda is usually defined inside functions.

### Capture List

- `[=]`: capture all local variables by value.
- `[&]`: capture all local variables by reference.
- `[var1, var2]`: capture only specified local variables.

Global variables can always be used in a lambda.

### Return Type

- Return type is `void` if there is no return statement.
- Return type may be inferred automatically when there is a return statement.
- Return type can also be explicitly specified using `-> return_type`.

Example:

```cpp
using namespace std;

int square(int n) {
  return n * n;
}

int main() {
  int x = 10;

  cout << square(x) << endl; // normal function call

  // lambda expression call
  cout << [](int n) { return n * n; }(x) << endl; // return type inferred as int

  return 0;
}
```

### Value vs Reference in Range-based `for`

```cpp
using namespace std;

int main() {
  vector<int> range = {1, 2, 7};

  for (int v : range) {
    // pass by value -> copy
    v += 10; // does not change range
  }

  for (int &v : range) {
    // pass by reference -> changing v changes range elements
    v += 10;
  }

  return 0;
}
```

### Lambda Example: Compute Max of Two Numbers

```cpp
using namespace std;

int main() {
  int x[3][2] = {{3, 6}, {9, 5}, {7, 1}};

  for (int k = 0; k < static_cast<int>(sizeof(x) / sizeof(x[0])); ++k) {
    cout << [](int a, int b) { return (a > b) ? a : b; }(x[k][0], x[k][1]) << endl;
  }

  return 0;
}
```

### Capture Required for Local Variables

```cpp
using namespace std;

int main() {
  int x = 10;

  cout << [=]() { return x * x; }() << endl; // OK

  return 0;
}
```

<span style="color: red;">反例: []() { return x * x; }(); （x is not captured）</span>

## Default Capture and Exception

```cpp
using namespace std;

int main() {
  int x = 1;
  int y = 2;
  int a = 3;

  [=, &x]() {
    cout << ++x << ", " << y << ", " << a << endl;
  }();

  return 0;
}
```

<span style="color: red;">反例: [&x, =]() { cout << ++x << ", " << y << ", " << a << endl; }(); （default capture must be first）</span>

Global variables can always be used in lambdas, and they are modifiable if not `const`.

## `mutable` Keyword

```cpp
using namespace std;

int main() {
  int x = 10;

  [=]() mutable { cout << ++x << endl; }(); // OK

  return 0;
}
```

<span style="color: red;">反例: [=]() { cout << ++x << endl; }(); （x is read-only without mutable）</span>

## `auto` Keyword

The keyword `auto` allows declaring variables without explicitly writing the type.
The type is inferred by the compiler.

```cpp
auto i = 42;        // int
auto d = 3.14;      // double
auto msg = "hello"; // const char*
```

## Function Pointer

```cpp
using namespace std;

bool fun() {
  return true;
}

int main() {
  bool (*ptr)() = nullptr;
  ptr = fun;

  cout << fun << endl; // function address
  cout << ptr() << endl;

  return 0;
}
```

### Choose Function by Condition

```cpp
using namespace std;

int larger(int a, int b) {
  return (a > b) ? a : b;
}

int smaller(int a, int b) {
  return (a < b) ? a : b;
}

int main() {
  int (*f)(int, int) = nullptr;

  int choice;
  cin >> choice;

  f = (choice == 1) ? larger : smaller; // dereference * is optional here

  cout << f(8, 3) << endl;
  return 0;
}
```

### Array of Function Pointers

```cpp
using namespace std;

int larger(int a, int b) {
  return (a > b) ? a : b;
}

int smaller(int a, int b) {
  return (a < b) ? a : b;
}

int main() {
  int (*f[2])(int, int);

  f[0] = larger;
  f[1] = smaller;

  cout << f[0](5, 9) << endl;
  cout << f[1](5, 9) << endl;

  return 0;
}
```
