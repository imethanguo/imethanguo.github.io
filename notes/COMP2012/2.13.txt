Const:

----------------------------------------------------------------------------

non-const obj               Okay                            Okay

------------------------------------------------------------------------------

const pointer:

Any const to the left of * refers to the object being pointed to
Any const to the right of * refers to the pointer itself


int * ptr 

Case1 -> const int* ptr
Case2 -> int const* ptr
"Any const to the left of * refers to the object being pointed to"
Case 1,2 -> Pointer to "constant int"

Case3 -> int* const ptr
"Any const to the right of * refers to the pointer itself"
-> Constant pointer to int ( " fixed ")

"Any const to the left of * refers to the object being pointed to"
"Any const to the right of * refers to the pointer itself"

Remark:
int* const * p = const (int*) * p
指向“ 常量int*” 的指针


const int * ptr;
int x = 10;
ptr = &x;

cout<< *ptr ; // Okay!

(*ptr) = 20; // Error! We regard (*ptr) as a constant int!

x++; // Okay , x is modifiable!

(ptr)++; // NO !
         // x cannot be modified through ptr , but can be modified through x itself


For "int* const", Initialization is a must!

int y = 20;
