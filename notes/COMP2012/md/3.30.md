## Operator Overloading Basics

Examples of operators:

- `<<` insertion (output) operator
- `+` addition operator
- `<` comparison operator

For user-defined class:

```cpp
class Vector {
public:
  Vector(double a = 0, double b = 0) : x(a), y(b) {}

  Vector operator+(const Vector& other) const {
    return Vector(x + other.x, y + other.y);
  }

private:
  double x, y;
};
```

Then `a + b` is equivalent to `a.operator+(b)`.

## Coercion

```cpp
int a = 2;
double b = 0.5;
// a + b: a is converted to double
```

With converting constructor:

```cpp
Vector a;
a + 10; // works: 10 -> Vector(10, 0)
```

<span style="color: red;">反例: 10 + a; （fails if operator+ is only a member of Vector）</span>

Reason: member operator needs class object on the left operand.

## Global Non-member `operator+`

```cpp
class Vector {
public:
  Vector(double a = 0, double b = 0) : x(a), y(b) {}

  double getX() const { return x; }
  double getY() const { return y; }

private:
  double x, y;
};

Vector operator+(const Vector& a, const Vector& b) {
  return Vector(a.getX() + b.getX(), a.getY() + b.getY());
}
```

Now both `a + 10` and `10 + a` can work through implicit conversion.

## Syntactic Sugar

- `a + b` is sugar for `operator+(a, b)` or `a.operator+(b)` depending on definition.

## Operators That Cannot Be Overloaded

- `.`
- `::`
- `?:`
- `.*`

You also cannot change an operator's:

1. Arity (number of operands)
2. Associativity
3. Precedence

Every overloaded operator must involve at least one user-defined type.

## Output Operator `<<`

`ostream` objects like `cout` should be passed and returned by reference.

```cpp
using namespace std;

class Vector {
public:
  Vector(double x = 0, double y = 0) : x_(x), y_(y) {}
  double getX() const { return x_; }
  double getY() const { return y_; }

private:
  double x_, y_;
};

ostream& operator<<(ostream& os, const Vector& v) {
  os << "(" << v.getX() << ", " << v.getY() << ")";
  return os;
}
```

Why global function? `cout << v` means left operand is `cout`, not `v`.

## Input Operator `>>`

```cpp
using namespace std;

class Vector {
public:
  void setX(double x) { x_ = x; }
  void setY(double y) { y_ = y; }

private:
  double x_{0}, y_{0};
};

istream& operator>>(istream& is, Vector& a) {
  double x, y;
  is >> x >> y;
  a.setX(x);
  a.setY(y);
  return is;
}
```

## `operator+=` Return Type

If return type is non-const reference:

```cpp
Vector& operator+=(const Vector& other);
(a += b) += b; // OK
```

If return type is const reference:

```cpp
const Vector& operator+=(const Vector& other);
```

<span style="color: red;">反例: (a += b) += b; （left side becomes const）</span>
